    Class GraphRepositoryMongo

    MongoDB implementation of GraphStore.

    • nodes in collection graph_nodes
    • edges in collection graph_edges

    Bidirectional adjacency is modeled by storing edges as documents with from and to. Neighbor queries look up edges where from==id OR to==id.
