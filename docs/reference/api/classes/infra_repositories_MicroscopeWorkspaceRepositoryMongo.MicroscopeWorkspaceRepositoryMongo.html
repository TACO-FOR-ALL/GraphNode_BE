    Class MicroscopeWorkspaceRepositoryMongo

    MongoDB를 저장소로 활용하는 Microscope 워크스페이스 리포지토리 구현체. Microscope 워크스페이스의 메타데이터와 각각의 문서 진행 상태를 저장하고 갱신합니다.

    updateDocumentStatus

    • updateDocumentStatus(
          groupId: string,
          docId: string,
          status: "COMPLETED" | "FAILED" | "PENDING" | "PROCESSING",
          sourceId?: string,
          graphPayloadId?: string,
          error?: string,
          session?: ClientSession,
      ): Promise<void>

      워크스페이스 내에 존재하는 특정 문서의 상태(PENDING/COMPLETED/등)를 업데이트합니다. SQS 워커가 AI 모델 처리를 마치고 결과를 줄당 시 이 메서드로 성공/실패 값을 갱신합니다.

      Parameters

      • groupId: string

        워크스페이스 UUID

      • docId: string

        문서를 식별하는 기준이 되는 문서 식별 ID (taskId)

      • status: "COMPLETED" | "FAILED" | "PENDING" | "PROCESSING"

        변경될 AI 처리 상태

      • OptionalsourceId: string

        (optional) Graph가 생성되었으면 Graph Node/Chunk와 연결하기 위한 sourceId

      • OptionalgraphPayloadId: string
      • Optionalerror: string

        (optional) 실패 시 반환받은 에러 메시지

      • Optionalsession: ClientSession

      Returns Promise<void>

      Throws

      MICRO_WORKSPACE_NOT_FOUND 대상 워크스페이스 또는 문서(docId)를 찾지 못할 때

      Throws

      MICRO_DOCUMENT_UPDATE_FAIL 배열 내부 요소를 수정하다가 실패할 시

      Remarks

      • MongoDB의 위치 연산자 ($)를 활용하여 documents 배열 내에서 id 매칭 대상 1건의 속성만 정확히 갱신합니다.
      • 상위 워크스페이스 및 하위 문서 양쪽의 업데이트 날짜(updatedAt)를 동시 기록합니다.

      Implementation of MicroscopeWorkspaceStore.updateDocumentStatus
