    • removeNodeCascade(userId: string, nodeId: number): Promise<void>

      특정 노드와 연결된 모든 엣지를 함께 삭제합니다. (Cascade) 이 메서드는 GraphService.deleteNode를 호출하며, 해당 서비스의 레포지토리 구현체에서 노드와 엣지를 트랜잭션처럼 처리하는 로직에 의존합니다.

      Parameters

      • userId: string

        작업을 요청한 사용자 ID

      • nodeId: number

        삭제할 노드의 ID

      Returns Promise<void>

      Promise

      Throws

      • DB 작업 중 오류 발생 시

      Example

      // 노드 5와 연결된 모든 엣지를 함께 삭제
      await service.removeNodeCascade('u-123', 5);
