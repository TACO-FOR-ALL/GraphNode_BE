    applyBatchNodes

    • applyBatchNodes(
          items: { nodePayload: GraphNode; vectorPayload: any }[],
      ): Promise<{ created: string[]; errors: any[]; vectorUpserted: string[] }>

      Apply a small batch of node+vector prepared payloads.

      This two-phase helper:

      • Phase A: attempts to persist nodes via GraphService (best-effort)
      • Phase B: attempts to upsert vectors for successfully created nodes

      The function returns detailed results so callers can implement retries or an outbox pattern when stronger guarantees are required.

      Parameters

      • items: { nodePayload: GraphNode; vectorPayload: any }[]

        Array of { nodePayload, vectorPayload } produced by prepareNodeAndVector

      Returns Promise<{ created: string[]; errors: any[]; vectorUpserted: string[] }>
