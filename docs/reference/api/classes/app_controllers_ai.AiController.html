    Class AiController

    AiController 클래스

    AI 기능과 관련된 모든 API 엔드포인트를 처리하는 컨트롤러입니다. 생성자 주입(Constructor Injection)을 통해 필요한 서비스들을 의존성으로 받습니다.

    참고: 메서드 내부에서 try-catch를 사용하지 않는 이유는, 라우터 정의 시 asyncHandler를 사용하여 에러 발생 시 자동으로 전역 에러 핸들러로 넘기기 때문입니다.
