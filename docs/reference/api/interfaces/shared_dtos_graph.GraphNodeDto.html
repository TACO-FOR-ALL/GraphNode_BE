    Remarks

    • id는 외부 AI 모듈이 부여한 정수 ID이며, 사용자마다 고유하다.
    • origId는 원본 대화(예: conversationId)를 나타낸다.
    • timestamp는 ISO 8601(UTC) 문자열 또는 null이다.
