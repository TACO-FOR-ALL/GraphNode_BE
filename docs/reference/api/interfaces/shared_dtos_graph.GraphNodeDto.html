    그래프 노드 DTO.

    Param: userId

    Param: sourceType

    노드의 출처 ('chat' | 'markdown' | 'notion')

    생성 시각(ISO 8601 UTC)

    Param: updatedAt

    갱신 시각(ISO 8601 UTC)

    Remarks

    • id는 외부 AI 모듈이 부여한 정수 ID이며, 사용자마다 고유하다.
    • origId는 원본 대화(예: conversationId)를 나타낸다.
    • timestamp는 ISO 8601(UTC) 문자열 또는 null이다.
