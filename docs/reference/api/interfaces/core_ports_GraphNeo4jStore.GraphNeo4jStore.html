    Interface GraphNeo4jStore

    GraphNeo4jStore (Neo4j)

    Neo4j Graph Database에 그래프 구조(Topology)를 저장하기 위한 인터페이스입니다. GraphDocumentStore(MongoDB)와는 달리 그래프 순회 및 관계 중심의 쿼리에 최적화된 인터페이스를 지향합니다.
