    Interface NoteRepository

    모듈: NoteRepository 인터페이스 책임: 노트 및 폴더 데이터의 영속성(Persistence) 계층 추상화.

    • MongoDB 트랜잭션 지원을 위해 쓰기 작업에 session 옵션을 포함한다.
    • 구현체는 src/infra/repositories에 위치한다.
